    概要

    HTML是我们进入WEB领域的第一扇门，所以最好先来了解一下什么是WEB标准 很多面试官会提问什么是W3C标准，W3C是一个社区,他是各类技术标准为各类应用的开发而打造的一个开放的Web平台。他是WEB标准规范化要求的提出者（看单词了解一下？World Wide Web Consortium） 那么WEB标准并不单单是一个标准，而是一个系列标准的集合,包括:

    • -结构标准 如 HTML,XML,XHTML

    • -表现标准 如 CSS

    • -行为标准 如 （W3C DOM）、ECMAScript等

    作为一个WEB前端开发者需要养成一种良好编码习惯，实际开发要以公司的规范为标准，所以如果是初学者的话最好先熟悉一下那些标准中的规范 请参考http://www.w3c.org
